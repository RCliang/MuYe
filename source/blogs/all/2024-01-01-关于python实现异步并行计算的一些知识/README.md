## 基本概念

1. 同步和异步：同步指程序按顺序执行，异步指程序可以同时执行，比如一个函数执行完之后再执行另一个函数，而另一个函数执行完之后再执行下一个函数，以此类推，这种执行方式我们一般称为同步。而异步指程序可以同时执行多个函数，比如一个函数执行完之后， another函数执行完之后，再执行第三个函数，以此类推，这种执行方式我们一般称为异步。
